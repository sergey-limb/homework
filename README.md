# Домашнее задание к занятию «Кеширование Redis/memcached», Самсонов СВ

### Задание 1. Кеширование

Приведите примеры проблем, которые может решить кеширование.

Ответ:
- Ограниченная пропускная способность сети - Кэш уменьшает количество сетевых запросов, экономя трафик и ускоряя ответ
- Высокая нагрузка на базу данных - Часто запрашиваемые данные (например, профиль пользователя или список категорий) можно хранить в кэше. Повторные запросы не нагружают базу.
- Повторная загрузка статических файлов (музыка, изображения, видео) - Кэш браузера позволяет пользователю не загружать те же файлы повторно, ускоряя загрузку сайта.
- Медленный ответ базы - Сложные SQL-запросы или аналитика выполняются один раз, результат сохраняется в кэше и отдается мгновенно при повторных запросах.
